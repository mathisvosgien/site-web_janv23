Cours 3 - L'insertion d'images en HTML.

Pour insérer une image en HTML, vous devez utiliser la balise <img>. Voici comment utiliser cette balise :

L'attribut ‘src’ (source) de la balise <img> doit être utilisé pour spécifier l'URL de l'image que vous souhaitez afficher. L'attribut ‘alt’ (alternatif) doit être utilisé pour fournir un texte alternatif qui sera affiché à la place de l'image si elle ne peut pas être chargée ou si elle n'est pas accessible pour des raisons de accessibilité.

Vous pouvez également utiliser d'autres attributs avec la balise <img> pour contrôler l'apparence de l'image, tels que ‘width’ et ‘height’ pour redimensionner l'image, ‘border’ pour ajouter une bordure autour de l'image, et ‘style’ pour appliquer des styles CSS à l'image.
